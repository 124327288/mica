# mica-swagger 组件

## 说明
对 `swagger` 和 `knife4j` 的封装，方便使用。

**注意**: 添加 `@Api` 注解的控制器会自动暴露给 `swagger`。

## 使用
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-swagger</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-swagger:${version}")
```
