# mica-springdoc 组件

## 说明
对 `springdoc-openapi` 和 `knife4j` 的封装，方便使用。

## 使用
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-springdoc</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-springdoc:${version}")
```
